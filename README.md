# Schema, vecka 45
###### Fullstack Webapplikation med TypeScript, vecka 1 av 7

## Introduktion

En introduktion till Typescript, vad det är och dess fördelar samt vad som behövs installeras för att kunna köra det. 
Vi tittar även på skillnaden på statiskt typat språk och ett dynamiskt typat språk och kodexempel på hur man använder statiska typer och interfaces TS.


## Mål för veckan:

* Förståelse för vad Typescript är och dess fördelar
* Kunna konfigurera upp och kompilera en TS-fil till en JS-fil och förstå varför en kompilator behövs
* Kunna använda sig av statiska typer i variabler
* Förstå skillnaden på types och interfaces
* Kunna hämta från ett API och typa upp API-svaret som ett interface.
